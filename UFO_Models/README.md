# UFO and FeynArts Model Files

Holds the UFO files for event generation

## Folders


* [UV_BSM_ToyModel_NLO-UFO](./UV_BSM_ToyModel-NLO-UFO): UFO folder used to generate the VLF events.

* [SMS_stop_NLO-UFO](./SMS_stop_NLO-UFO): UFO folder used to generate the Scalar events.

* [DMsimp_s_spin1](./DMsimp_s_spin1): UFO folder used to generate the $Z'$ events.
